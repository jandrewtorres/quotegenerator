# RandQuote https://randquote.herokuapp.com/
## Built With:
- html5
- css
- javascript/jquery
- Quotes on Design API [Link to API](https://quotesondesign.com/api-v4-0/)

## Features
- random quotes using Quotes on Design API
- responsive layout for mobile and browser
- tweet quote button
- Deployed on heroku
